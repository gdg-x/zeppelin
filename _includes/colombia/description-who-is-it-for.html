¿Eres una mujer interesada en aprender cómo construir aplicaciones web con las últimas tecnologías? Tenemos buenas noticias para ti: ¡Tendremos un taller de un día para principiantes! Se llevará a cabo el Sábado,11 de Noviembre, en Yuxi Global,Centro empresarial ciudad del rio, Medellín.

Creemos que la industria de TI se beneficiará mucho de traer más mujeres a la tecnología. Queremos darte una oportunidad para aprender a programar y convertirte en una programadora.

Durante el taller, construirás y desplegarás tu primera aplicación web Angular es una plataforma de código abierto para la creación de aplicaciones web, desarrollada por Google y la comunidad. Ofrece diversas herramientas para crear fácilmente una aplicación dinámica e interactiva que puede ejecutarse en navegadores de escritorio, móviles e incluso robots.

Para participar en el taller necesitarás algunos conocimientos básicos en desarrollo web: HTML, CSS y JavaScript básico o cualquier otro lenguaje de programación. Si eres un principiante total, aquí hay algunos recursos que puedes mirar.

El taller no tiene costo. Sólo llena el formulario de Inscripción.

No esperes demasiado - puedes aplicar hasta el 6 de Noviembre, 2017!
