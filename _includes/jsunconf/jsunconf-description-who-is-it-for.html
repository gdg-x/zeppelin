Are you a woman interested in learning how to build web applications with the latest technologies? We have good news for you: We are holding a workshop for beginners! It will take place during JS Unconf on Saturday, March 25th, 2017 at BMK school Hamburg.

During the workshop you will build and deploy your very first Angular Web Application! Angular is an open-source platform for building web application, developed by Google and the community. It gives various tools to easily create a dynamic, interactive app that can run on desktop browses, mobile and even robots!

To participate in the workshop you'll need some basic knowledge in web development: HTML, CSS and basic JavaScript or any other programming language. If you're a total beginner, no problem. Here are some resources to catch up.

The workshop is free of charge. Just fill out the application form.
